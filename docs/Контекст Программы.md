**Залы Upper и принцип работы программ**

В зале Upper всегда идут две программы одновременно, которые ведёт один тренер.

**Комбинации:**

- Push + Pull  
    <br/>
- Full Body + Upper  
    <br/>
- Full Body + Arms  
    <br/>
- Upper + Arms  
    <br/>
- Upper + Upper  
    <br/>

❗ Это обязательное правило: в слоте зала Upper никогда не бывает одной программы — только две параллельные.

**Каждая программа состоит из 8 упражнений.  
**Структура формата:

- 8 станций;  
    <br/>
- каждая станция — отдельное упражнение;  
    <br/>
- станции в рамках двух программ независимы друг от друга (две отдельные маршрутизации).  
    <br/>

**На каждой станции — по 2 одинаковых тренажёра.**

- На одной станции одновременно работают 2 человека.  
    <br/>
- Работа индивидуальная, НЕ в парах.  
    <br/>
- Общее распределение: каждый атлет занимает свою рабочую точку.  
    <br/>

**Вместимость одной программы Upper:**

Формула: 8 упражнений × 2 тренажёра = 16 человек.

То есть:

- Push = 16 человек  
    <br/>
- Pull = 16 человек  
    <br/>
- Upper = 16 человек  
    <br/>
- Full Body = 16 человек  
    <br/>
- Arms = 16 человек  
    <br/>

Не зависит от комбинации.

**Общая вместимость зала Upper:  
**две программы × 16 человек = 32 человека.

Это полностью соответствует документу, реальной практике и текущей логике маршрутизации.

**Как работает распределение (пример Push + Pull):**

- Push: 8 станций × 2 тренажёра → 16 атлетов.  
    <br/>
- Pull: 8 станций × 2 тренажёра → 16 атлетов.  
    <br/>

Каждая из двух программ — это отдельный круг, со своими 8 станциями.

Силовая тренировка на всё тело.  
Целый комплекс упражнений, направленный на проработку всех основных групп мышц за одну тренировку. Цель — обеспечить равномерное развитие мышц, повысить общую силу и выносливость и улучшить функциональные способности. Тренировка подходит для людей с разными уровнями подготовки.

**Пример плана Full Body:**

**Принцип работы зала Legs:**

1.  В зале Legs всегда идут две разные программы одновременно, которые ведёт один тренер: Legs и Glute.

2.  Каждая программа состоит из 5 упражнений:  
    Legs = 5 станций, Glute = 5 станций.

    Формула: 5 упражнений × 3 человека = 15 человек.

**Светозвуковой синхронизатор**

Все программы в силовых залах Upper, Legs, Full Body проходят в сопровождении свето-звукового синхронизатора.

Светозвуковой синхронизатор — это система, которая управляет ритмом групповых тренировок. Она задаёт длительность работы, отдыха и переходов с помощью световых и звуковых сигналов. Благодаря этому все атлеты и тренеры работают в едином темпе. Синхронизатор обеспечивает корректную маршрутизацию и синхронное выполнение нескольких программ одновременно (в Upper — 2 программы, в Legs — 2 программы, в Full Body Villa — 3 программы). Это позволяет тренеру сосредоточиться на технике и безопасности, а атлетам — уверенно двигаться по тренировочному кругу.

**Синхронизатор выполняет 2 ключевые задачи:**

1.  **Управление временем тренировочного цикла  
    <br/>**

Он чётко задаёт:

- когда начинается работа;  
    <br/>
- когда заканчивается;  
    <br/>
- когда начинается отдых;  
    <br/>
- когда начинается переход;  
    <br/>
- длительность раундов;  
    <br/>
- длительность интервалов.  
    <br/>

Это позволяет:

- держать одинаковый темп у всех;  
    <br/>
- избежать хаоса;  
    <br/>
- облегчить работу тренера;  
    <br/>
- обеспечить отработку методологии.  
    <br/>

1.  **Синхронизация сразу нескольких программ в одном зале  
    <br/>**

Особенно важно для:

- Upper (2 программы одновременно);  
    <br/>
- Legs (2 программы одновременно);  
    <br/>
- Full Body в Villa (3 программы одновременно).  
    <br/>

Синхронизатор делает так, чтобы все три программы (Push + Pull + Legs или Upper + Arms + Glute) работали:

- в одном темпе;  
    <br/>
- в одном ритме;  
    <br/>
- с одинаковой длительностью интервалов;  
    <br/>
- со скоординированными переходами.  
    <br/>

Без этого невозможно провести 2–3 тренировки параллельно на 32 или 48 человек. Это полностью убирает путаницу и позволяет атлетам двигаться по залу уверенно.

**Как он работает:**

Синхронизатор — это комбинация:

1.  **Световых индикаторов  
    <br/>**

- Зелёный → конценрическая фаза, при которой атлет поднимает вес.  
    <br/>
- Красный → эксцентрическая фаза, при которой атлет плавно под напряжением опускает вес.  
    <br/>

1.  **Звуковых сигналов  
    <br/>**

Система подаёт:

- стартовый сигнал (начало упражнения);  
    <br/>
- специфический сигнал концентрической фазы;  
    <br/>
- специфический сигнал эксцентрической фазы;  
    <br/>
- сигнал окончания подхода.  
    <br/>

Громкость и тип звука оптимизированы, чтобы быть слышимыми во всём зале.

Без синхронизатора:

Синхронизатор — это единая система управления залом.

Например, в Villa (Full Body) в одном слоте идут: Push, Pull, Legs — все три должны:

- стартовать одновременно;  
    <br/>
- заканчивать одновременно;  
    <br/>
- переходить одновременно.

Иначе зал не функционирует.

- интенсивности;

Без синхронизатора все тренировки стали бы разными — методология развалится.

Синхронизатор позволяет тренеру:

- сосредоточиться на технике;

То есть тренер не следит за таймером — он следит за качеством, атмосферой и безопасностью.

**METCON**

Тренировка на улучшение общей физической формы.

Меткон (сокращенно от metabolic conditioning) развивает выносливость, силу и скорость. Тренировка сочетает в себе высокоинтенсивное кардио с силовыми нагрузками, что помогает эффективно сжигать калории (до 1 000 ккал за одну тренировку), улучшать метаболизм и повышать выносливость. Основной фокус меткона — работа на пределе своих возможностей с короткими периодами отдыха. Подходит для людей с разными уровнями подготовки.

Меткон делится на:

- POWER — преимущественно силовые упражнения;  
    <br/>
- CARDIO — преимущественно комплексные кардио;  
    <br/>
- CORE — преимущественно упражнения на мышцы кора.  
    <br/>

**BOOTCAMP**

Тренировка направлена на улучшение сердечно-сосудистой системы и приведения мышц в тонус.

Упражнения укрепляют сердце и приводят мышцы в тонус. Здесь и работа на беговой дорожке, и работа со свободными весами, что повышает выносливость, скорость и силу. Буткамп помогает улучшить общую физическую форму и функциональные возможности и эффективно сжигает калории (до 1 000 ккал за одну тренировку). Подходит для людей с разными уровнями подготовки.

Bootcamp делится на:

- Full Body — тренировка всего тела;  
    <br/>
- Legs & Shoulders — тренировка ног и плеч;  
    <br/>
- Chest & Biceps — тренировка груди и бицепса;  
    <br/>
- Back & Triceps — тренировка спины и трицепса;  
    <br/>
- Core — тренировка мышц кора;  
    <br/>
- Stretching — тренировка на улучшение техники бега и развитие гибкости всего тела.  
    <br/>

**Reshape**

Тренировочная система на основе функционального пилатеса, адаптированного под современные запросы тела.

Проводится на реформерах. Работа на тренажере с регулируемой нагрузкой — за счёт пружин и интенсивности работы на подвижной платформе, в дополнении работа с ремнями — активно включаются стабилизаторы, глубокие мышцы корпуса, мышцы-антагонисты.

**Главные задачи Reshape:**

- Улучшить контроль над телом и осанку.  
    <br/>
- Укрепить мышечный корсет, особенно глубокие мышцы, которые почти не работают на классических силовых/интервальных тренировках.  
    <br/>
- Повысить метаболизм и общий расход энергии за счёт работы мышц, а не только пульса.  
    <br/>
- Замедлить возрастные изменения: сохранить подвижность суставов, эластичность тканей, силовую выносливость.  
    <br/>
- Помочь атлету “переподключиться” к телу — координация, дыхание, ощущение сегментов, контроль амплитуды.

  

**Assessment**  — это  диагностический протокол Hero’s Journey, который состоит из **четырёх этапов** и используется для определения стартового уровня атлета, построения персонального маршрута и отслеживания прогресса.

**Assessment  — структура**

**1\. InBody  
**Измерение состава тела:

- масса тела,  
    <br/>
- мышечная масса,  
    <br/>
- жировая масса,  
    <br/>
- процент жира,  
    <br/>
- висцеральный жир.  
    Проводится **до любой нагрузки**, чтобы получить чистые показатели.  
    <br/>

**2\. Functional Test  
**Тест на функциональную выносливость:

- **40 секунд максимальное количество burpee.  
    **Показывает общий уровень кондиции и адаптацию к интенсивной работе.  
    <br/>

**3\. Strength Test  
**Силовой круг из трёх станций:

- **Leg Press**,  
    <br/>
- **Chest Press**,  
    <br/>
- **Horizontal Row.  
    **Режим — 10 / 8 / 5 повторений с ростом веса,  
    интервал 50 секунд работа / 50 секунд отдых.  
    Цель — зафиксировать **реальный максимум** для расчёта силы и прогрессии.  
    <br/>

**4\. Cardio Test**

- **6-минутный бег** на максимальную дистанцию.  
    Используется для оценки аэробной выносливости и динамики VO₂ max.  
    <br/>

&nbsp;**Цель Assessment** 

Получить **комплексную картину состояния атлета**:

- состав тела,  
    <br/>
- функциональный уровень,  
    <br/>
- силовой потенциал,  
    <br/>
- кардиорезерв.  
    <br/>

Результаты служат базой для назначения **маршрута тренировок Hero’s Journey** и контроля прогресса на каждом этапе.

  
<br/><br/><br/>

**1\. Текущее состояние: как сейчас составляются программы**

Сейчас **все тренировочные планы** для залов составляются **вручную экспертом**.

Это особенно сложно для залов, где **одновременно идут несколько программ**:

- в зале **Upper** всегда 2 программы (например, Push + Pull, Full Body + Upper и т.д.)  
    <br/>
- в зале **Legs** всегда 2 программы (Legs + Glute)  
    <br/>
- в зале **Full Body (Villa)** сразу 3 программы (Push + Pull + Legs / Upper + Arms + Glute и т.д.)  
    HJ асистент (3)  
    <br/>

При этом эксперт должен одновременно учитывать:

1.  **Не пересекать тренажёры между программами, идущими в одном зале  
    <br/>**
    - Если в зале Upper в одном слоте идут Push и Pull, то упражнения/станции этих программ **не должны использовать одни и те же тренажёры**, чтобы не было накладок при маршрутизации.  
        <br/>
    - Та же логика в Legs (Legs + Glute) и Full Body (3 программы).  
        <br/>
2.  **Сохранить научную и методическую логику каждой программы  
    <br/>**
    - Правильное распределение нагрузок по мышечным группам  
        <br/>
    - Баланс базовых и изолирующих движений  
        <br/>
    - Контроль объёма (подходы/повторы), интенсивности и плотности  
        <br/>
    - Логика прогрессии от тренировки к тренировке/неделе  
        <br/>
3.  **Обеспечить разнообразие планов из большого ассортимента упражнений  
    <br/>**
    - Из имеющегося парка тренажёров, свободных весов и станций нужно собирать **максимально разнообразные, но при этом методически грамотные** планы.  
        <br/>
    - Нельзя, чтобы одни и те же связки повторялись слишком часто, иначе атлеты «закисают» на одних и тех же упражнениях.  
        <br/>

Итог:  
Каждый новый план — это ручная «головоломка», где эксперт одновременно решает:

- конфликты по оборудованию  
    <br/>
- логику нагрузки  
    <br/>
- разнообразие  
    <br/>
- особенности каждого зала и связки форматов  
    <br/>

Это **очень энергозатратно**, плохо масштабируется на сеть студий и сильно завязано на одном-двух людях.

**2\. К чему вы хотите прийти: автоматизация создания тренировочных планов**

Цель — прийти к системе, которая:

1.  **Автоматически генерирует тренировочные планы** для залов Upper, Legs, Full Body (и дальше — для MetCon/Bootcamp/Reshape и т.д.)  
    <br/>
2.  **Гарантирует отсутствие пересечений по оборудованию** между параллельными программами в одном зале (например, Push и Pull не борются за один и тот же тренажёр в текущем слоте).  
    <br/>
3.  **Соблюдает научную методологию**:  
    <br/>
    - структура занятия для каждого формата (Push, Pull, Upper, Legs, Glute, Arms, Full Body…)  
        <br/>
    - нужный объём и интенсивность  
        <br/>
    - баланс движений (тяги/жимы, горизонтальные/вертикальные, односторонние/двусторонние, квадрицепс/ягодицы/задняя поверхность и т.д.)  
        <br/>
4.  **Создаёт разнообразие**:  
    <br/>
    - система не «застревает» на одном и том же наборе упражнений  
        <br/>
    - внутри недели/месяца тренировочные планы отличаются, но остаются методически выверенными  
        <br/>
5.  **Масштабируется на всю сеть Hero’s Journey**:  
    <br/>
